# P - D.D.L - Private Direct Download Link

## Local Setup
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5000`

## Docker Setup
1. `docker build -t p-ddl .`
2. `docker run -p 5000:5000 p-ddl`

## IP/Domain Setup
1. Deploy to a server.
2. Set `PORT=5000`.
3. Use a reverse proxy like Nginx.
